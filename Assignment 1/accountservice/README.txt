Account Domain-service

- Run app.py
- Use OpenAPI to use the microservices

Microservices include:
- Account
	- Create an account
		- Input:
			- Name
			- Address
			- Birthdate
			- Email
			- Password
	- Get account information
		- Input:
			- Email
			- Password
	- Delete an account
		- Input:
			- Email
			- Password
- Login
	- Update the password of an account
		- Input:
			- Email
			- Old Password
			- New Password
	- Retreive password if forgotten
		- Input:
			- Unique account id
			- Email
	- Retreive email if forgotten
		- Input:
			- Unique account id
			- Password
- Settings
	- Update the account settings
		- Input
			- Email
			- Password
			- Verification Code Preference
			- Email Notification Preference
			- Promotion Notification Preference
			- Invoice Notification Preference
			- Browser Notification Preference
